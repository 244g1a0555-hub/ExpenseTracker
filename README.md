Expense Tracker

An Android Expense Tracker App built using Kotlin and Jetpack Compose.

Features

- Income Tracking
- Expense Tracking
- Categories (Food, Travel, Shopping, Education)
- Expense History
- Delete Expense
- Date & Time Tracking
- Firebase Integration
- Dynamic Pie Chart Statistics
- Dark Mode

Technologies Used

- Kotlin
- Jetpack Compose
- Firebase Firestore
- Android Studio
